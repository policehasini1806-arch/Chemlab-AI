# ChemLab AI 🧪

An AI-powered interactive chemistry learning platform featuring virtual labs, simulations, and intelligent tutoring — built with Google AI Studio and Gemini.

🔗 **Live Demo:** [chemlab-ai-933497967070.asia-southeast1.run.app](https://chemlab-ai-933497967070.asia-southeast1.run.app)

---

## About the Project

ChemLab AI is an educational web application designed to make chemistry learning more engaging and accessible. It combines AI-generated content with interactive simulations to give students a hands-on learning experience without the need for a physical lab.

The platform was conceptualized, built, and refined using Google AI Studio and Gemini, and deployed on Google Cloud Run.

---

## Features

- 🔬 **Virtual Labs** — Simulate chemistry experiments interactively in the browser
- 🤖 **AI Tutoring** — Get instant explanations and answers to chemistry questions powered by Gemini
- 📊 **Simulations** — Visualize chemical reactions, molecular structures, and lab processes
- 📚 **Topic Coverage** — Covers core chemistry concepts suitable for school and undergraduate students
